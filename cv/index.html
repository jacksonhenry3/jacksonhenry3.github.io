Consultant, Concord Consortium

Concord, MA : 2011-2012

I worked with the Concord Consortium, a nonprofit scientific education company. I worked on the next gen molecular workbench project to check and see if the simulation of radial bonds they were implementing was correct. I examined the javascript code and extracted from it frequency data from several different simulations. I then took this data and plotted it using python and found a discrepancy between the expected frequency (that of the same simulation in the java version of MW) and the found frequency. This lead them to correct several parameters and fix the simulation. I also created a setup guide for Ubuntu so others running the OS could participate in the project.

Accomplishments

  • Found error in current simulation
  • Created getting started guide.
